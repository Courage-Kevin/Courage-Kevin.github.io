Java基础知识总结

发表于 2018-02-05 | 分类于 Java基础知识

java基础知识总结

1. Java编译运行过程

  1. 编译期：java源文件，经编译生成.class文件

  2. 运行期：JVM加载.class文件，并运行class

  3. 特点：跨平台，一次编译到处使用

2.名词释义

  1. JVM：java虚拟机，作用就是加载.class文件，运行.class文件

  2. JRE： java开发环境，除了包含JVM外，还包含java程序运行所必须的环境

  3. JDK：java开发工具包，除了包含JRE外，还包含编译，运行等命令工具3.

3.数据类型

byte类型 8位 范围：-128~127
short 类型 16位 范围：-32768~32767
char 类型 16位 范围：\u0000~\u65535
int类型 32位 范围：-2147483648~2147483647
long 类型 64位 范围：很大很大 2^63
float 类型 32位
double 类型 64位
boolean 类型 8位

4.基本类型自动转换

  • 从小到大依次为：byte < short < int < long < float < double
  • 自动类型转换：小类型转大类型 （小杯子往大杯子倒水）
  • 强制类型转换：大类型转小类型 （大杯子往小杯子倒数）

5.运算符

  1. 算数： +，-，*，/，%，++，–
  2. 关系：>,<,>=,<=,==,!=,boolean
  3. 逻辑：&&，||，!boolean
  4. 赋值：=，+=，-=，*=，/=，%=
  5. 字符连接运算符：+
  6. 条件/三目运算：boolean?数1:数2;

6.分支结构

  1. If
  2. If…else
  3. If…elss if
  4. Switch…case

7.循环

  1. while循环
  2. do…while循环
  3. for循环

8.数组

  1. 数组的定义：int [ ] arr =new int [ 10 ]

  2. 数组的初始化：int [ ] arr = new int [ ]

  3. 数组的访问：通过数组名.length可以获取数组的长度，也就是数组内元素的个数通过索引来访问数组元素，下标从0开始，最大到数组长度-1

  4. 数组遍历

9.访问修饰符

private 同类可见 可以修饰变量、方法 注意：不能修饰类(外部类)
default 同包同类可见 可以修饰类、接口、变量、方法
protected 同包同类和所有子类可见 可以修饰变量、方法 注意不能修饰类(外部类)
public 所有类可见 可以修饰类、接口、变量、方法

10.非访问修饰符

为了实现某一些功能，java也提供了许多非访问修饰符

  1. static修饰符：被static修饰的变量和方法叫做静态变量和静态方法

    • 静态变量：static关键字用来声明独立于对象的静态变量，无论一个类实例化多少对象，他的静态变量只有一份拷贝

    • 静态方法：static关键字用来声明独立于对象的静态方法，静态方法不能使用类的非静态变量(静态方法只能使用类的静态变量)

  1. final修饰符：

    • final变量：不能被重新赋值

    • final方法：final修饰的方法可以被子类继承，但是不能被子类修改

    • final类： final类不能被继承，没有类能够继承final类的任何特征

  1. abstract修饰符：

    • 抽象类：抽象类不能用来实例化对象，目的：是为了将来对该类进行扩充。

    • 抽象方法：继承抽象类的子类必须实现父类的所有抽象方法，除非这个子类也是抽象类，

    • 注意：如果一个类包含若干个抽象方法，那么该类必须声明为抽象类，抽象类不可以包含抽象方法。

  1. 抽象加单总结：

    ​ 1）抽象方法的意义：在于将方法的设计和方法的实现分离。

    ​ 2）抽象类需要被继承，抽象类定义的抽象方法子类必须重写。

    ​ 3）抽象类不能创建对象，但是可以包含构造方法，成员属性，普通方法。
